## Summary

[RAW-BOOK:5] Visual framework showing pricing relationships: spot bonds, forward bonds/repo, interest rate swaps, and options. Spot-forward linked via net carry (repo cost - coupon income). Swap linked to bond pair (fixed + floating). Options priced via volatility + spot/forward. Options relate to 3 corners via put-call parity, moneyness, sensitivity. Core insight: if you can hedge it, you can price it.
