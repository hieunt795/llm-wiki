## Summary

[RAW-BOOK:5] Binary option pays fixed sum if exercised ITM (irrespective of payoff depth), not spot-dependent payoff. Types: at-expiry (European, pays if ITM at maturity), one-touch (American, pays at first strike hit), no-touch (pays if strike never hit). Path-dependent (one-touch, no-touch) cost ~2× European. Premium quoted as % of payout (e.g., 10% premium = 10% cost for 100% payout).
